<h3>📌 POS Weekly Compliance Dashboard</h3>

<p><b>Problem:</b><br/>
Inconsistent transaction reporting and charge discrepancies across agents.
</p>

<p><b>Approach:</b></p>
<ul>
  <li>Consolidated weekly datasets into a monthly report</li>
  <li>Built pivot tables for agent-level summaries</li>
  <li>Applied conditional formatting to detect anomalies</li>
  <li>Designed dashboard for monitoring</li>
</ul>

<p><b>Impact:</b></p>
<ul>
  <li>Improved compliance visibility</li>
  <li>Reduced manual audit effort</li>
  <li>Enabled faster issue detection</li>
